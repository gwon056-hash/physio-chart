MOA Charting install/update popup
버전: 2026.07.05-4

추가 기능
- 홈 화면에 설치 버튼 팝업 구현
  - iPhone: Safari 설치 안내 팝업
  - Android: beforeinstallprompt 지원 시 실제 설치창 호출
  - Android 설치창을 사용할 수 없으면 브라우저별 설치 안내
  - 이미 설치된 경우 설치 완료 상태 안내
  - PC 설치 안내

- 업데이트 확인 팝업 구현
  - 현재 버전 표시
  - 최신 버전 표시
  - 최신 상태 안내
  - 새 버전이 있으면 '지금 업데이트' 버튼 제공

유지 기능
- 자동 업데이트
- 기존 localStorage 키 chartingPWA.v1
- 기존 저장 차팅
- 이전 차팅 검색
- 선택 삭제
- 백업/복원
- 모든 평가 데이터
